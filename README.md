# Cypto Tracker

Crypto Tracker is a script written in python for real-time price tracking for traders.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```

## Usage

```python
python app.py
```
